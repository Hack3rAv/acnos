# Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

Participants are expected to:

- Be respectful and considerate
- Use welcoming and inclusive language
- Respect different viewpoints
- Accept constructive criticism gracefully

## Unacceptable Behavior

The following behavior will not be tolerated:

- Harassment or discrimination
- Personal attacks
- Trolling or insulting comments
- Publishing private information without permission

## Enforcement

Project maintainers have the right to remove or restrict participants who violate this Code of Conduct.
