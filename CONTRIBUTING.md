# Contributing to ACNOS

Thank you for your interest in contributing to **ACNOS (Avrodip Community Network Operating System)**.

ACNOS is a community-driven project aimed at building decentralized communication infrastructure.

---

## How to Contribute

You can contribute in several ways:

* Improving documentation
* Suggesting new features
* Reporting bugs
* Proposing architectural improvements
* Helping design the ACNOS Mesh Protocol (AMP)

---

## Contribution Process

1. Fork the repository
2. Create a new branch

Example:

```
git checkout -b feature-name
```

3. Make your changes
4. Commit changes

```
git commit -m "Added feature"
```

5. Push to your fork

```
git push origin feature-name
```

6. Create a Pull Request

---

## Guidelines

When contributing:

* Keep code clean and readable
* Document new features
* Follow the architecture of ACNOS
* Avoid breaking existing functionality

---

## Areas Where Help Is Needed

* Mesh networking architecture
* Hardware node design
* IoT integration
* Drone communication systems
* Network routing algorithms
