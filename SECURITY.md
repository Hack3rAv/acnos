# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **ACNOS**, please report it responsibly.

Do NOT publicly disclose vulnerabilities before they are reviewed.

Instead contact the maintainer.

---

## Reporting Method

Send a detailed report including:

* Description of the issue
* Steps to reproduce
* Potential impact
* Suggested fix (if available)

---

## Contact
